# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-01

### Added
- Initial project setup with Android project structure
- Strict code quality enforcement with Git hooks and GitHub Actions
- Pre-commit hooks for linting, building, and testing
- Version bump validation and automatic version management
- GitHub Actions workflows for PR checks, version bumping, and release management
- Enforcement lock system to protect enforcement files
- Test lock system for TDD approach
- Basic Android app structure with MainActivity
